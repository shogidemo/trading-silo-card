# 穀物サイロカードコレクション ユーザーマニュアル

## はじめに

「穀物サイロカードコレクション」は、クイズに答えながら穀物の世界について楽しく学べるカードコレクションアプリです。サイロ、穀物、商社に関する知識を身につけながら、カードを集めていきましょう。

![トップページ](/screenshots/01-home.png)

## カードの種類

### サイロ（Silo）
穀物を貯蔵する巨大な施設のカードです。日本各地にある実在のグレーンターミナルについて学べます。
- 関東グレーンターミナル（茨城県）
- 東北グレーンターミナル（青森県）
- 日本サイロ千葉事業所（千葉県）
- 東洋グレーンターミナル（愛知県）
- 博多港サイロ（福岡県）

### 穀物（Grain）
世界を支える農産物のカードです。それぞれの穀物の特徴、産地、用途、栄養素などを学べます。
- 小麦
- 大豆
- トウモロコシ
- 米
- 大麦

### 商社（Trader）
穀物を世界中に届ける企業のカードです。穀物貿易に関わる商社について知ることができます。

## カードのレアリティ

カードには3段階のレアリティがあります：
- **Common（コモン）**: 基本的なカード
- **Rare（レア）**: やや珍しいカード
- **Legendary（レジェンダリー）**: 最も希少なカード

## 使い方

### 1. クイズに挑戦する

トップページの「クイズに挑戦する」または「クイズで学ぶ」ボタンをクリックすると、カテゴリ選択画面が表示されます。

![クイズカテゴリ選択](/screenshots/02-quiz-select.png)

3つのカテゴリ（サイロ・穀物・商社）から好きなものを選んでタップしてください。

### 2. クイズに回答する

カテゴリを選択すると、クイズが出題されます。4つの選択肢から正解だと思うものを選んでタップしてください。

![クイズ出題画面](/screenshots/03-quiz-question.png)

### 3. 結果を確認する

回答後、正解・不正解が表示され、解説を読むことができます。正解すると新しいカードを獲得できます。

![クイズ結果画面](/screenshots/04-quiz-result.png)

- **正解の場合**: 関連するカードを獲得！「カードを見る」で獲得したカードを確認できます
- **不正解の場合**: 解説を読んで知識を深めましょう

### 4. コレクションを確認する

トップページの「コレクション」ボタンをクリックすると、獲得したカードの一覧が表示されます。

![コレクション画面](/screenshots/05-collection.png)

- カテゴリボタンでフィルタリングできます（すべて/サイロ/穀物/商社）
- カードをタップすると詳細情報が表示されます
- 未獲得のカードはシルエット表示になります

### 5. 進捗を確認する

トップページでは以下の情報を確認できます：
- **Collected**: 獲得したカード枚数
- **Total Cards**: 全カード枚数
- **Accuracy**: クイズの正答率
- **Collection Progress**: 全体の収集進捗（パーセント）
- 各カテゴリごとの収集状況

## 画面の説明

| 画面 | 説明 |
|------|------|
| トップページ | 全体の進捗状況が一目でわかるダッシュボード |
| クイズページ | カテゴリ選択 → クイズ出題 → 結果表示 の流れ |
| コレクション | カードギャラリーと収集統計 |

## データの保存について

- 獲得したカードやクイズの成績は、ブラウザのローカルストレージに自動保存されます
- 同じブラウザで再度アクセスすれば、前回の続きから遊べます
- ブラウザのデータを削除すると、進捗もリセットされます

## 動作環境

- モダンブラウザ（Chrome、Firefox、Safari、Edge）推奨
- スマートフォン、タブレット、PCいずれでも利用可能
- レスポンシブデザイン対応
